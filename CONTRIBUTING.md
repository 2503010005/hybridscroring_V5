# Contributing to HybridScoring_V5

Thank you for considering contributing to HybridScoring_V5.

This project focuses on developing AI-assisted educational technologies for Indonesian Automated Essay Scoring and Educational Decision Support Systems.

---

## How to Contribute

### Reporting Issues

Please use GitHub Issues to report:

- bugs,
- unexpected behaviors,
- documentation problems,
- enhancement suggestions.

When reporting an issue, include:

- operating system,
- Python version,
- steps to reproduce,
- expected behavior,
- actual behavior.

---
